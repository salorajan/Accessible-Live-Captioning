# Accessible Live Captioning (ALC)

A lightweight, highly accessible Windows live captioning application. It transcribes audio in real-time from either your system microphone or speakers (loopback) using an offline local Whisper model, displaying screen-reader-friendly captions and optionally reading them aloud.

Developed by **Robert Danaraj** (salorajan@gmail.com).

---

## Getting Started

Refer to [manual.md](manual.md) for detailed setup instructions, configuration settings, and guides on preventing audio feedback loops when using screen readers.

### Quick Start:
*   **Desktop GUI (wxPython):** Double-click [run_app.bat](run_app.bat) to launch.
*   **Console Companion (CLI):** Double-click [run_cli.bat](run_cli.bat) to launch in terminal.
